# 분류 기준 적용 사례

## 📋 실제 민원 분류 예시

### 사례 1: 악취 민원
```
신청인: 박철수
주민등록번호: 900315-1234567
제목: 근처 화학공장에서 나오는 악취 문제
내용: 
거주 지역 근처에 위치한 화학공장에서 매일같이 악취가 발생하고 있습니다. 
주민들의 일상생활과 건강에 심각한 악영향을 미치고 있으니 
즉시 대책을 요청합니다.

분류 프로세스:
1️⃣ 키워드 추출
   - 매칭된 키워드: "악취", "냄새", "화학공장", "공장"
   
2️⃣ ML 모델 분석
   - 부처 신뢰도: 92% (환경부)
   - 부서 신뢰도: 88% (대기환경과)
   - 전체 신뢰도: 90%
   
3️⃣ 분류 근거
   - 법적근거: 대기오염 예방, 악취 관리 (환경부 직제 제2조)
   - 정책근거: 미세먼지 저감 정책 강화 (정부 보도자료 2024.01)
   - 결론: 환경부 > 대기환경과

✅ 최종 분류: 환경부 / 대기환경과 (신뢰도: 높음)
```

### 사례 2: 도로 손상 민원
```
신청인: 이영희
주민등록번호: 950520-2345678
제목: 아파트 앞 도로의 심각한 포장 손상
내용:
아파트 앞 도로가 패인 상태로 6개월 이상 방치되어 있습니다.
자동차 통행에 위험을 초래하고 있으며, 
특히 비오는 날씨에는 물이 고여 더 위험합니다.
빠른 보수를 요청합니다.

분류 프로세스:
1️⃣ 키워드 추출
   - 매칭된 키워드: "도로", "포장", "패인도로", "손상"

2️⃣ ML 모델 분석
   - 부처 신뢰도: 94% (국토교통부)
   - 부서 신뢰도: 91% (도로과)
   - 전체 신뢰도: 92.5%

3️⃣ 분류 근거
   - 법적근거: 도로 건설, 유지보수, 포장 관리 (국토교통부 직제 제5조)
   - 정책근거: 도로 안전 강화 정책 (정부 보도자료 2024.03)
   - 결론: 국토교통부 > 도로과

✅ 최종 분류: 국토교통부 / 도로과 (신뢰도: 높음)
```

### 사례 3: 불법 주차 민원
```
신청인: 김민준
주민등록번호: 920830-3456789
제목: 아파트 입구의 계속되는 불법 주차
내용:
아파트 입구 도로에 차량이 불법 주차되어 
응급차량 통행과 주민 출입에 방해가 되고 있습니다.
계속해서 같은 차량이 주차하는 상황이 반복되고 있습니다.

분류 프로세스:
1️⃣ 키워드 추출
   - 매칭된 키워드: "주차", "불법주차", "위반"

2️⃣ ML 모델 분석
   - 부처 신뢰도: 88% (경찰청)
   - 부서 신뢰도: 89% (교통경찰과)
   - 전체 신뢰도: 88.5%

3️⃣ 분류 근거
   - 법적근거: 도로 교통 단속, 주차 위반 관리 (경찰청 직제 제3조)
   - 정책근거: 교통안전 사고 예방 강화 (정부 보도자료 2024.07)
   - 결론: 경찰청 > 교통경찰과

✅ 최종 분류: 경찰청 / 교통경찰과 (신뢰도: 높음)

⚠️ 반복민원 감지:
- 동일인 확인: 김민준
- 과거 민원 검색: 3건 발견 (유사도 65%, 78%, 72%)
- 결과: 반복민원 알림 발송 [중간 수준]
```

### 사례 4: 경계 케이스 - 도로 환경오염
```
신청인: 박지은
주민등록번호: 930615-4567890
제목: 도로변 불법 폐기물 투기로 인한 환경오염
내용:
도로변 주차장 근처에 산업폐기물이 불법으로 투기되어 있으며,
이로 인해 악취가 발생하고 주변 환경이 오염되었습니다.
빠른 대처를 요청합니다.

분류 이슈:
- 도로 문제인가? (국토교통부 도로과)
- 환경오염 문제인가? (환경부 대기환경과)

분류 프로세스:
1️⃣ 키워드 추출
   - 도로 관련: "도로변", "도로"
   - 환경 관련: "폐기물", "투기", "환경오염", "악취"

2️⃣ ML 모델 분석
   - 국토교통부: 45%
   - 환경부: 78%
   - 환경부 > 대기환경과: 82%

3️⃣ 분류 근거
   - 주요 문제: 환경오염 및 불법 폐기물 투기
   - 도로 자체의 손상은 없음
   - 환경 관련 부처가 최우선

✅ 최종 분류: 환경부 / 대기환경과 (신뢰도: 중간)
```

### 사례 5: 신뢰도 낮은 경우
```
신청인: 정수미
주민등록번호: 960722-5678901
제목: 정부 정책의 전반적인 개선 필요
내용:
최근 정부의 여러 정책들이 국민 생활에 충분히 반영되지 않고 있습니다.
전반적인 정책 개선을 강력히 요청드립니다.

분류 프로세스:
1️⃣ 키워드 추출
   - 매칭된 키워드: 없음 (명확한 부처 지칭 없음)

2️⃣ ML 모델 분석
   - 부처별 확률이 고르게 분포 (각 20%)
   - 부서별 확률이 불균형 분포
   - 전체 신뢰도: 35%

3️⃣ 분류 근거
   - 법적근거: 불명확
   - 정책근거: 불명확
   - 결론: 구체적 지칭 부서 없음

✅ 최종 분류: 환경부 / 대기환경과 (신뢰도: 낮음)
⚠️  주의: 관리자 수동 검토 권장
```

## 🔍 분류 신뢰도별 처리 절차

### 신뢰도 80% 이상 (높음)
```
민원 접수
    ↓
자동 분류
    ↓
[신뢰도 80% 이상]
    ↓
자동 할당 ✅
    ↓
담당 부서 자동 배정
해당사항 없음 (즉시 처리 진행)
```

### 신뢰도 60~80% (중간)
```
민원 접수
    ↓
자동 분류
    ↓
[신뢰도 60-80%]
    ↓
예비 할당
    ↓
관리자 확인 화면 표시
    ↓
관리자 검토 후 확정 또는 재분류
```

### 신뢰도 60% 미만 (낮음)
```
민원 접수
    ↓
자동 분류
    ↓
[신뢰도 60% 미만]
    ↓
공개 대기 상태
    ↓
관리자가 수동으로 할당하거나
관련 부처 협의를 통해 분류
```

## 📊 분류 정확도 개선

### 모델 재학습 조건

1. **월간 재학습**: 최근 100건의 실제 분류 결과 포함
2. **정책 변경 시**: 부처 조직개편 또는 정책 변경 즉시
3. **오류율 증가 시**: 신뢰도 낮은 민원 증가 감지 시
4. **새로운 카테고리**: 미분류 민원 증가 감지 시

### 피드백 루프

```
민원 분류
    ↓
담당자 처리
    ↓
실제 담당 부처와 비교
    ↓
일치 ✅ → 성공 카운트 증가
    ↓
불일치 ❌ → 오류 분석
    ↓
모델 재학습 대상 수집
    ↓
월간 배치 재학습
```
